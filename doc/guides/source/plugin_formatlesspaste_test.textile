h2. Testing the Formatless Paste Plugin

endprologue.

h3. Step 1

Enable the formatless paste plugin in the boilerplate page.

h3. Step 2

Configure the formatless paste plugin according to the description in the guides page.

h3. Step 3 

Load the boilerplate and paste content from word with formatings you have configured to be stripped.
e.g. add some italic text and configure "i" to be stripped.

h3. Step 4

Check the pasted results. Everything went as expected if the configured formatings have been removed.
e.g. if the italic text is in your editable but is not italic any longer
