Welcome to WYSIWHAT: A semantic HTML5 editor for textbooks, books, and open education resources

WYSIWHAT is a repo for the development of Aloha plugins, models, tests, and examples for use in the production of books and open educational resources. The changes focus on editing semantically marked up documents in HTML5. We intend to commit code back to Aloha for plugins that are general purpose (like mathematics editing and enhanced table editing) when they reach maturity.

Contributors

  • OERPUB


    OERPUB is not a single project but a collective where open source projects plan and produce tools and architectures that work best for authoring, adapting, remixing, and publishing open education resources and then delivering them to the web, mobile, tablet, and print. Kathi Fletcher is a Shuttleworth Foundation Fellow contributing designs and implementations of a semantic editor based on Aloha that is integrated into a document system that imports from many formats and publishes to oer repositories (currently Connexions).

  • Connexions
    • Connexions (cnx.org) is developing an editor here for Connexions modules. It is mostly the same as the oerpub editor, with different styling.
  • Booktype
    • Booktype is a project of Sourcefabric. Booktype is also developing a wysiwhat editor for their Booktype 2 platform to be released next year.
  • Siyavula
    • Siyavula is creating free and open textbooks for South Africa. They have contributed user interface designs to the effort and will be collaborating on specializing the editor for their textbook sprints.
